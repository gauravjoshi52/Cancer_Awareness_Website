Project Overview:
A responsive and modern website for Cancer Awareness & Support organization featuring a landing page with inspirational quotes, contact form, and comprehensive information about their mission.

Features:
Responsive Landing Page - Works perfectly on all devices

Hero Banner - Eye-catching banner with overlay text

Real-time Quotes - Fetches inspirational quotes from public API with fallback

Contact Form - Fully functional contact form with validation

Modern Design - Clean, professional UI with gradient colors

Smooth Navigation - Easy-to-use navigation with smooth scrolling

File Structure:
cancer-awareness-website/
|--- index.html          # Main website page
|--- styles.css          # All styling and responsive design
|---script.js            # Interactive functionality

Technologies Used:
HTML5 - Semantic structure

CSS3 - Modern styling with CSS variables and Flexbox/Grid

JavaScript - Interactive features and API integration

Quotable API - For inspirational quotes

Quick Start:
Method 1: Direct File Opening
Download all 3 files into the same folder

Open index.html in your web browser

The website will work immediately

API Integration:
The website uses the Quotable API (https://api.quotable.io/random) to fetch inspirational quotes. If the API is unavailable, it gracefully falls back to a curated list of quotes.
